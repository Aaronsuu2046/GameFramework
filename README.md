**這是一個PAIA遊戲的模板，依照此模板所開發的遊戲，可以在PAIA－Desktop透過ＡＩ來玩遊戲，並進行ＡＩ競賽。**

# 使用方式
1. create a new folder
2. copy and past MyGame
3. refactor code in MyGame
